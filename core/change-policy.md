# Change Policy

## Local experiment
A pattern used in one project only.

## Candidate pattern
A pattern reused enough to consider promotion.

## Operating standard
A pattern promoted into `core/`, `skills/`, or templates.

## Promotion rule
Promote only when reuse is proven and the added standard reduces confusion.

## Deprecation rule
Deprecate before removing when changing a core concept.

## Update rule
When a core concept changes, update impacted layers together.
