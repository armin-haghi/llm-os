# Minimal Templates

## docs/project-spine.md
- What: product, system, or workflow in one sentence
- Why: reason this work matters now
- Who: user, buyer, operator, or internal owner
- Constraints: real boundaries on scope, platform, compliance, or timing
- Success criteria: observable outcome that makes the project worth continuing
- Current milestone: active stage and milestone link
- Canonical links: only durable sources of truth

## docs/current-milestone.md
- Snapshot: project, stage, milestone, owner, status, date
- Why this milestone matters: what it unlocks or de-risks now
- In scope: outputs required for acceptance
- Out of scope: tempting work that should stay out
- Acceptance criteria: observable checks for done
- Dependencies: external inputs required to complete or review
- Blockers: current blockers only
- Open questions to clarify during active sessions: uncertainty the team can
  reduce while working

## docs/open-questions.md
- Question: unresolved decision or uncertainty
- Why it matters: what changes if this is answered differently
- Who should resolve it: build agent, review agent, or human
- Current default assumption: temporary working assumption in use now
- Decide by: date, milestone gate, or review checkpoint

## docs/session-brief.md
- Snapshot: project, stage, milestone, owner, date
- Project summary: shortest useful reload of current truth
- Next action: first concrete move for the next serious session
- Blocking question: single biggest question that can stop progress
- Latest decisions: only decisions that changed the work
- Current risks: risks that can change milestone outcome
- Expected output from this session: durable artifact, evidence, or clarity gain

## docs/review-report.md
- Snapshot: project, stage, milestone, reviewer, date
- Summary: short review conclusion
- Result: pass, conditional pass, or fail
- Evidence checked: concrete evidence reviewed
- Unresolved assumptions: what review could not verify
- Risks: remaining risk after review
- Required human decisions: only decisions the human must make
- Recommended next step: the best move after review
