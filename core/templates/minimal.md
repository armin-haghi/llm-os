# Minimal Templates

This file remains as a compatibility index.

The canonical operational templates now live in `templates/` as one file per
artifact so agents can load only the template they need:
- `templates/project-spine.md`
- `templates/current-milestone.md`
- `templates/open-questions.md`
- `templates/session-brief.md`
- `templates/review-report.md`

Use `templates/README.md` for the template-layer overview.
