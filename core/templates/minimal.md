# Minimal Templates (Canonical)

This file defines the single default artifact surface.

## Default surface: `docs/current-milestone.md`

Use this template as the canonical working document for an active milestone.

```md

## Metadata
- Milestone:
- Owner (Human):
- Status:
- Target date:
- Links: Notion / Issue / PR

## Milestone objective

## Success criteria

## Stages
### 1) Scope
- Decisions:
- Constraints:

### 2) Plan
- Approach:
- Sequence:

### 3) Build
- Implementation status:

### 4) Review
- Findings:
- Required changes:

### 5) Ship / Track
- Release/status notes:
- Next milestone trigger:

## Task board (stage -> lane)
- [ ] Task — Stage: Scope | Lane: Human | Owner: | Notes:
- [ ] Task — Stage: Plan | Lane: Review agent | Owner: | Notes:
- [ ] Task — Stage: Build | Lane: Build agent | Owner: | Notes:

## Open risks / blockers

## Latest update
- Date:
- Summary:
- Next actions:
```

## Optional artifacts
Use additional docs only when the milestone surface is no longer sufficient.
Legacy templates remain compatibility-only.
